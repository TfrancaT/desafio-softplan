Sobre o Núcleo

Nossa missão é promover o acesso à tecnologia e ao conhecimento digital

Oferecer cursos de tecnologia gratuitos de alta qualidade, promovendo a inclusão digital e reduzindo a lacuna tecnológica.

Preparar os participantes para carreiras na área de tecnologia.

Fomentar a inovação e o empreendedorismo tecnológico.

Nossos Valores:

  • Servir a comunidade.
  • Promover aprendizado contínuo.
  • Desenvolvimento Pessoal.
  • Levar o digital para todos.

Cursos oferecidos:

Informática Básica

Aprenda o essencial da informática, incluindo uso de computadores, navegação na web e noções de segurança online.

Programação Web

Crie sites interativos usando HTML, CSS e JavaScript, aprendendo a construir páginas web atraentes.

Programação Mobile

Desenvolva aplicativos para dispositivos móveis (iOS e Android) com as linguagens Swift, Kotlin ou JavaScript, preparando-se para o mercado de desenvolvimento móvel.
